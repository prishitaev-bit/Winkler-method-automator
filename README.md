# Engineering Chemistry Lab Automator 🧪⚙️

## Overview
A terminal-based C utility designed to automate repetitive post-titration calculations for first-year B.Tech chemistry laboratory experiments. This tool eliminates manual calculation errors and speeds up data processing for experiments such as determining Dissolved Oxygen (Winkler's Method) and estimating Ion Exchange Resin capacity.

## Features
* **Automated Averaging:** Accepts multiple burette readings and processes the concordant values.
* **Instant Concentration Output:** Calculates final concentrations (e.g., mg/L for dissolved oxygen) based on user-inputted sample volume and titrant normality.
* **Error Reduction:** Built with basic data validation to prevent negative volume inputs.

## Tech Stack
* **Language:** C
* **Interface:** Command Line / Terminal

## Why I Built This
During chemistry lab practicals, processing titration formulas manually across multiple trials can lead to calculation errors and wasted time. I developed this program to apply logical programming concepts to a real-world engineering workflow.
